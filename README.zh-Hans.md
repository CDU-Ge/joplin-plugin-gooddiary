# GoodDiary 插件

GoodDiary 是一个 Joplin 插件，它可以每天自动为你打开日记。

## 功能

- 自动创建每日笔记
- 自动创建每日笔记文件夹
- 支持多语言（英语、简体中文、繁体中文）

## 安装

1. 下载插件包并解压到 Joplin 插件目录。
2. 在 Joplin 中启用插件。

## 使用

插件会在启动时自动创建当天的日记，并每隔两小时检查并创建新的日记。

## 配置

插件会根据 Joplin 的全局语言设置自动选择对应的语言。
